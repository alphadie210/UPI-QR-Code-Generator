# UPI-QR-Code-Generator

A Python project to generate UPI payment QR codes for multiple apps including **PhonePe, Paytm, and Google Pay**.  
This tool allows you to create QR codes based on a user's UPI ID and view them as images using the Pillow library.
---
## ✨ Features
- Generate QR codes for different UPI apps (PhonePe, Paytm, Google Pay).
- Save QR codes as image files (`.png`).
- Display QR codes directly using Pillow.
- Simple and customizable payment URL format.
---
## 📦 Requirements
- Python 3.x
- [qrcode](https://pypi.org/project/qrcode/) library
- [Pillow](https://pypi.org/project/Pillow/) library

Install dependencies:
```bash
pip install qrcode pillow
